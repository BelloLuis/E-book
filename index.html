Lyrics Note

Lyrics Note es una app móvil pensada para músicos y aficionados que quieren encontrar letras y notas de canciones en un formato claro para tocar. La interfaz es simple, con navegación intuitiva y animaciones suaves para una experiencia agradable.

UX Writing Diseño de icono SQLite Patrón de navegación

Identidad

El logo combina la “L” (Lyrics) y la “N” (Note) en un símbolo simple y memorable.

Ícono

Versión optimizada para pantallas móviles y tiendas de aplicaciones.

Inicio

Barra de búsqueda, accesos rápidos y mensajes guía para nuevos usuarios.

Buscador

Vista de canción

Editor

Guardado

Cuenta

Registro sencillo para personalizar la experiencia y guardar favoritos.

Base de datos

Implementada con SQLite (Canciones.db). Tabla “Canciones” con _id, título, autor y letra.

Enfoque: almacenamiento local y lectura rápida.
QR List

QR List es un sistema web para tomar asistencia con códigos QR. Estudiantes crean su perfil y generan su código; docentes lo escanean y el sistema registra nombre, fecha y hora en la base de datos.

HTML/CSS/JS PHP MySQL Autenticación por rol QR workflow
